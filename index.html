ProFam PETase Variant Analysis

ProteinMPNN scoring and experimental validation of 15 computationally designed PETase variants generated by ProFam

Overview

We designed 14 PETase variants using ProFam (protein family-conditioned language model) and included Kubu-PETase (profam_5009) as a positive control. All 15 variants were experimentally tested by Adaptyv Bio for:

  • Expression in cell-free system (quantified by fluorescence, LOD = 2.5 µM)
  • pNPA esterase activity (slope of absorbance change)
  • PET hydrolysis (UV absorbance of released products)

Each variant was structurally modeled with both ColabFold (AlphaFold2) and AlphaFold3, then scored with ProteinMPNN to assess sequence-structure compatibility.

First Residue Substitution Experiment

To test MPNN score sensitivity, we substituted the first residue (Met) with each of the other 19 amino acids across all 15 AF3 structures and re-scored with ProteinMPNN. Each subplot compares the distribution of scores with wildtype M (blue) vs the substituted amino acid (red) across all 15 variants. The Δ value shows the change in mean score.

Result: All substitutions produced negligible score changes (Δ < 0.02), confirming that ProteinMPNN is largely indifferent to the identity of the first residue — consistent with N-terminal flexibility and lack of structured backbone contacts at position 1.

Methods

Variant Generation

14 PETase variants were generated using ProFam, a protein family-conditioned language model. Variants were sampled from the model's learned distribution over PETase-family sequences. Kubu-PETase (profam_5009) was included as a positive control, being a known functional PETase.

Structure Prediction

  • ColabFold: AlphaFold2 predictions using MMseqs2 for MSA generation. Rank-001 structures used for MPNN scoring.
  • AlphaFold3: Predictions using the AlphaFold3 server. Model-0 structures used for MPNN scoring.

ProteinMPNN Scoring

Each variant was scored using ProteinMPNN in --score_only mode with --seed 42. The score represents the mean negative log probability of each amino acid given the backbone structure. Lower scores indicate better sequence-structure compatibility.

Experimental Testing

All variants were expressed in a cell-free system and tested by Adaptyv Bio:

  • Expression: Quantified by fluorescence (LOD = 2.5 µM)
  • pNPA Assay: General esterase activity measured by p-nitrophenyl acetate hydrolysis (absorbance at 405 nm)
  • PET Assay: PET film hydrolysis measured by UV absorbance of released terephthalic acid products

Data Processing

Values below limit of detection (LOD) were treated as censored data. Binary classifications (expressed/not, active/not) were used for visualization. Correlations computed using Pearson and Spearman coefficients (scipy).
